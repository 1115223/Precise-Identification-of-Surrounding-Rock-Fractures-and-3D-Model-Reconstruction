# 基于钻孔图像的裂隙轮廓提取、粗糙度分析与 JRC（Joint Roughness Coefficient）计算

## 📖 Overview

本项目实现了**钻孔图像中裂隙轮廓的自动提取与粗糙度分析**，通过图像预处理、裂隙分割、骨架提取、PCA
坐标变换、多种采样策略及正弦曲线拟合，实现裂隙几何特征和 **JRC**
的自动计算。

### ✨ Features

-   自动读取数据集
-   自动解压 ZIP 数据
-   图像预处理（灰度化、CLAHE）
-   Otsu 自动阈值分割
-   裂隙骨架提取（Skeletonization）
-   PCA 主方向校正
-   三种采样策略（等 u、等弧长、曲率自适应）
-   Z² 与 JRC 自动计算
-   正弦曲线拟合
-   自动生成 CSV 与可视化结果

## 📐 JRC Calculation

计算轮廓参数：

\\\[ Z_2=`\sqrt{\\frac1n\\sum\\left(\\frac{dy}{dx}\\right)^2}`{=tex}
\\\]

经验公式：

\\\[ JRC=51.85Z_2\^{0.6}-10.37 \\\]

------------------------------------------------------------------------

## 🌊 Sine Curve Fitting

模型：

\\\[ y=R\\sin\\left(\\frac{2\\pi u}{P}+\\beta\\right)+C \\\]

输出参数：

-   振幅 R
-   周期 P
-   相位 β
-   中心线 C
-   拟合优度 R²
